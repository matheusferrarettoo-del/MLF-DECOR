MLF Decor - Site Catálogo

PASSO A PASSO:

1) Instale Firebase CLI:
   npm install -g firebase-tools

2) Login:
   firebase login

3) Dentro da pasta do projeto:
   firebase init
   - Escolher Hosting
   - Usar projeto existente: mlf-decor
   - Public folder: public
   - Single Page App: YES

4) Deploy:
   firebase deploy

Seu site ficará em:
https://mlf-decor.web.app
